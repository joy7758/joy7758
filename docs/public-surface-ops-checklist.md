- The profile README is the front door.
- `digital-biosphere-architecture` is the canonical interpretive entry point.
- The five canonical layer repositories should be visible in order.
- Demo and benchmark repositories should be secondary surfaces, not the front-door spine.

3. `agent-intent-protocol`
