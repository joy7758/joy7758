# Bin Zhang

Protocol-oriented research for multimodal and agentic AI systems.

Usually working on portable persona objects, executable agent objects, and evidence layers that are machine-checkable.

Current threads:

* [Persona Object Protocol (POP)](https://github.com/joy7758/persona-object-protocol)  
  A lightweight public draft protocol for portable persona objects in multimodal AI systems.
* [Agent Object Protocol (AOP)](https://github.com/joy7758/agent-object-protocol)  
  A draft protocol for portable agent objects and executable AI object structures.
* [ARO Audit](https://github.com/joy7758/aro-audit)  
  A lightweight audit and provenance layer for AI execution integrity.
